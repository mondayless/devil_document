### Jevil.camera

Open camera for taking picture or video

- Jevil.camera(cameraOption, callback)



#### Example code
```javascript
Jevil.camera({
    hasPicture : true,
    hasVideo : true,
    minSec : 3,
    maxSec : 10,
    ratio : 1.0
},
function(result){

})
```
